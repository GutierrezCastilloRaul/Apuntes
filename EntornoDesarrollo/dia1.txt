    ##### COMANDOS #####

    ¡¡ IMPORTANTE !! -> Todos los comandos empiezan con "git"

    - git commit -> crea un commit, un conjuto de cambios al repositorio. Se pueden "desincronizar"
        dando lugar a las ramas (branches)
    
    - git branch -> crea una ramificación, NO te lleva a ella, solo la crea, se tiene que 
        "commitear" por separado entrando en la rama correspondiente

        + branches + -> una rama, o "branch" suele tener otro color a las demás diferenciandolas así
            entre sí y de sus contenidos.

    - git checkout -> te mueve a ti y a la rama creada que tengas seleccionada al 
        repositorio a elegir. EJ: git checkout C1 : nos lleva a nosotros y el archivo que tengamos
        seleccionado o "encima" a un commit llamado "C1" (por el juego de la web)

    ### - BRANCHES PRO MAX ### -> Branches tmb se pueden utilizar para mover archivos a cualquier commit
        gracias al comando -f ( -f implica que se va a hacer una fuerza)
            · primer parametro: archivo a mover
            · segundo parametro: commit al que va destinado
    
        EJ:   _git branch -f main C2_
