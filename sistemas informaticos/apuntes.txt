---CREAR/NAVEGAR PROGRAMAS---

    .sh -> crea un programa de editor de texto

    nano (programa).sh -> crea un programa
        -> dentro de el debes poner *#!/bin/bash* para que entienda que es codigo.
        -> _echo "contenido"_ funciona como un print al ejecutal el programa crerado
        -> chmod 777 (programa) da permisos de administrador sobre el programa
        -> ./(programa).sh ejecuta el programa
